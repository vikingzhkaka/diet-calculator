# 每日饮食计算器（量化健身版）

一个单文件、可离线使用的每日宏量营养素（蛋白质 / 碳水 / 脂肪）计算器，专为力量训练人群设计。

## 功能
- **训练日 / 非训练日**一键切换：碳水随日型自动调整（训练日 4.5g/kg，非训练日 2.8g/kg）
- **体重自适应**：输入体重后，所有食物份量按比例缩放
- **宏量汇总 + 目标对比**：实时显示实际摄入 vs 目标，并给出达成度
- **食物替换**：同类食物下拉替换（肉 / 鱼 / 蛋 / 主食 / 蔬菜），克重与蛋碳脂实时联动
- **三套预设**：🏠 居家标准 / ✈️ 出差外食（真实点单话术）/ 🦵 周末腿训
- **本地持久化**：参数与替换选择存于浏览器 localStorage，刷新不丢

## 用法
直接用浏览器打开 `index.html` 即可，无需服务器、无需联网。

## 营养框架
基于《硬派健身》与《NSCA 运动营养指南》：
- 蛋白质 1.6–1.8 g/kg 体重
- 训练日碳水 4.5 g/kg，非训练日 2.8 g/kg
- 脂肪 0.7 g/kg

> 数值为通用参考，个体差异请咨询专业营养师或教练。
